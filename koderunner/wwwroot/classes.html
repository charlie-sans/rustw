KodeRunner Core Classes

Core

Central configuration and version management class.

GetVersion()

Returns the current version of KodeRunner.

Key Properties

  • RootDir: Base directory for KodeRunner files
  • CodeDir: Directory for project code
  • BuildDir: Directory for build outputs

RunnableManager

Manages the loading and execution of runnable components.

LoadRunnables()

Loads all available runnables from assemblies.

ExecuteFirstMatchingLanguage()

Executes the first runnable that matches the specified language. this also does partial matching so make sure to get your users to fully type qulify the name

BuildProcess

Handles build process execution and management.

SetupCodeDir()

Sets up the required directory structure.

SendInput()

Sends input to the build process.

TerminalCodeParser

Parses and converts terminal codes.

ParseToResonite()

Converts ANSI codes to HTML color codes.
