Welcome to KodeRunner

KodeRunner is a versatile project designed to manage and execute code snippets in various programming languages. It provides a structured environment to build, run, and manage code projects efficiently.

Key Features

  • Supports multiple programming languages.
  • Handles build processes and execution of code.
  • Provides a web interface for code management.
  • Integrates with WebSockets for real-time communication.

Project Structure

The project is organized into several key components:

  • Core: Contains the core logic and configuration settings.
  • Provider: Manages settings and configurations for runnables.
  • RunnableManager: Loads and executes runnable code snippets.
  • BuildProcess: Handles the build process for various languages.
  • TerminalProcess: Manages terminal input and output.

How to Use

To start using KodeRunner, follow these steps:

  1. Clone the repository to your local machine.
  2. Navigate to the project directory and build the project using dotnet build.
  3. Run the project using dotnet run.
  4. Access the web interface at http://localhost:5000.

Configuration

Configuration settings are managed through environment variables and a configuration file located at koderunner/Config/config.json. Ensure to set up your environment variables correctly before running the project.
